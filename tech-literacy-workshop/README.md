# Family Tech Literacy Workshop Project

Core promise:
> We help families raise tech-literate kids who can think clearly about systems, automation, data, interfaces, and AI—without screens taking over their lives.

## Format (current spec)
- 4-week series
- 75 minutes each
- Parent + child required (no drop-off)
- Minimal screens (no open internet; no kid accounts; presenter-controlled screen allowed)

(See: `ops/event_spec.md`.)
