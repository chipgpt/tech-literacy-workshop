# Family Tech Literacy Workshop Project

Core promise:
> We help families raise tech-literate kids who can think clearly about systems, automation, data, and interfaces—without screens taking over their lives.

Format:
- 4-week series
- 75 minutes each
- Parent + child required (no drop-off)
- Minimal screens (single presenter-controlled screen allowed)
- Faith-neutral

Folder map:
- curriculum/ — session outlines, facilitator scripts
- handouts/ — printable take-home PDFs/docs (rules sheets, vocab, plans)
- marketing/ — flyer copy, landing page copy, FAQs
- ops/ — venue checklist, materials list, run-of-show, pricing, signup/payment
- research/ — venue leads, competitor notes, references
- assets/ — images, icons, slides
