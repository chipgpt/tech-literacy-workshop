# Family Tech Literacy Workshop — Program Overview (4 weeks)

**Core promise:** Help families raise tech-literate kids who can think clearly about systems, automation, data, interfaces, and AI—without screens running the home.

**Audience:** Families with kids ~4–10 (older kids can do “stretch” prompts)  
**Format:** Parent + child required (no drop-off)  
**Duration:** 4 sessions × 75 minutes  
**Group size:** Designed for 10 kids max (plus parents); scalable with support staff  
**Screens:** Minimal (no open internet, no kid accounts; any screen use is presenter-controlled)  

---

## Program outcomes

### Kids will:
- Understand: technology is **tools made by people** to do a job.
- Learn to spot “pull” patterns (autoplay, notifications, infinite scroll).
- Practice a simple stop skill: **Pause → Ask → Choose**.
- Learn kid-safe AI basics: AI can look smart, but it’s pattern-making—**not magic**.

### Parents will:
- Use a calm, values-first framework instead of fear/shame.
- Learn simple mental models:
  - **Tool → Job**
  - **Input → Rule → Output**
  - **Data / Accounts / Defaults**
  - **Interface patterns → Boundaries**
- Leave with tangible family deliverables (cards/one-page plans) to reduce day-to-day conflict.

---

## The 4-week arc (what each week adds)

### Week 1 — Tools Made by People (Technology ≠ Magic)
**Big idea:** Technology is a tool made by people to do a job.

**Key concepts:** tool, job, choice, people-built, (optional) interface

**Activities:**
- Tool / Toy / Both sorting
- “Who made this?” chain (company → designers/engineers → interface choices)

**Family deliverable:** **Family Tech Values Card**
- “We use tech for…” (3 values)
- “Red lines…” (3 boundaries)
- Default rule: “Ask a parent before new apps/accounts.”

**Home move:** Put the card on the fridge and use it once.

---

### Week 2 — Systems & Automation (“Smart” Can Be Simple)
**Big idea:** Systems are **Input → Rule → Output**. Automation is rules running on repeat.

**Key concepts:** system, input, rule, output, automation, default

**Activities:**
- Create paper “If/Then” automations (one silly, one real)
- Translate common behaviors (autoplay, notifications, streaks) into input/rule/output

**Family deliverable:** **Family Automation Plan**
- 1 automation we want (reduce friction)
- 1 automation we want to turn off (reduce pull)
- 1 default rule for settings changes

**Home move:** Try one automation once.

---

### Week 3 — Data & Accounts (What’s Collected + What’s Remembered)
**Big idea:** Data is information. Accounts help tools remember you. Defaults shape what happens.

**Key concepts:** data, account, default, private

**Activities:**
- Keep / Share / Ask First sorting (data types)
- “Account machine” explanation: why apps ask for sign-in and what changes when they remember you

**Family deliverable:** **Family Account Rules**
- Ask before creating accounts
- Ask before entering personal info
- Parent-managed passwords
- (optional) purchase rule, device parking rule

**Home move:** Use one rule once.

---

### Week 4 — Interfaces, Habits, and the AI Reveal (Design Shapes Behavior)
**Big idea:** Interfaces are “handles” designed to guide behavior. Habits + environment beat willpower.

**Key concepts:** interface, pattern, boundary, habit, AI (as pattern-making)

**Activities:**
- Pattern spotting (autoplay, infinite scroll, recommendations, notifications, streaks)
- Boundary design sprint (choose 3: time/place/friction)

**Family deliverable:** **Family Tech Plan (final)**
- Values (3)
- Defaults (3 boundaries)
- Kid rule: Pause → Ask → Choose
- One-sentence enforcement script

**Capstone (8–10 min): The AI Reveal (kid-safe)**
- Bridge: Weeks 1–4 are the building blocks of AI
- Definition: **AI = patterns learned from lots of examples (data) + rules + outputs**
- Demo (recommended): “Human AI” paper demo (no screens)
- Family AI rules: Ask first; no personal info; treat answers as suggestions + double-check

**Home move:** Run one boundary for one week (no mid-week changes), then adjust.

---

## Instructor deliverables (what families physically leave with)
- Week 1: Family Tech Values Card
- Week 2: Family Automation Plan
- Week 3: Family Account Rules
- Week 4: Family Tech Plan (final) + AI rules

---

## Notes on delivery & facilitation

**Non-negotiables:**
- Parent + child together
- No open internet
- No kid accounts
- Keep it calm, practical, and shame-free

**Facilitation style:**
- Short instructions → activity → quick share-out
- Redirect venting back to the models (job; input/rule/output; data; patterns)
