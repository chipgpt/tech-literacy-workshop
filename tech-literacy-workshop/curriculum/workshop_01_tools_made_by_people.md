# Workshop 1 (75 min): Technology = Tools Made by People

**Theme:** Nothing is magic; technology is made by people on purpose.

**Audience:** Families with kids ages ~4–10

**Format rules:** Parent + child required. Minimal screens. (No open internet, no kid accounts.)

## Outcomes
### Kids will be able to:
- Explain “technology is a tool made by people to do a job.”
- Name a few everyday technologies and what job they do.
- Recognize that tools aren’t “good or bad” by default; use matters.

### Parents will be able to:
- Use a calm, values-first framing when kids ask about apps/devices.
- Ask: “What job is this tool doing for us?” and “Is it doing that job well?”

## Materials
- Printed **Tool / Toy / Both** cards (or real objects)
- Sticky notes + markers
- Poster paper / whiteboard
- Handout: **Family Tech Values Card** (draft)

## Run of show (75 minutes)
### 0–5 min — Welcome + norms
- “This is not daycare — parent and child do this together.”
- Norms: kind, curious, no shaming.
- The big idea: tools serve the family; screens don’t run the home.

### 5–15 min — Concept story: Tools do jobs
- Kid-level definition: “Technology is a tool made by people to do a job.”
- Prompt: “What tools do we use at home?”
- Introduce the word **interface** lightly: “Buttons/menus are the ‘handle’ for using the tool.”

### 15–40 min — Activity 1: Tool / Toy / Both sorting
- Families sort cards into 3 piles.
- Ask each family to pick 1 card and answer:
  - “What job does this do?”
  - “Who is it for (kid/parent/both)?”
  - “When does it become ‘too much’?”
- Older kids extension: write one sentence per card: “This tool helps with ___.”

### 40–55 min — Activity 2: “Who made this?” chain
- Pick 3 examples (YouTube, iPad, remote, Alexa, game console, etc.).
- Trace the chain:
  - “An app is made by a company.”
  - “A company hires designers/engineers.”
  - “Those people choose what buttons exist and what happens next.”
- Point: “If something feels pushy, that might be a design choice — not your kid being ‘bad’.”

### 55–70 min — Family deliverable: Tech Values Card
Families fill out:
- **Our family uses tech for:** (3 values)
- **Red lines:** (3 boundaries)
- **Default rule:** “Ask a parent before new apps / new accounts.”

Facilitator prompts (examples):
- Values: learning, creativity, connection, rest, kindness
- Red lines: no devices in bedrooms; no secret accounts; no autoplay on weekdays; no YouTube without parent

### 70–75 min — Close + Family Move
- “Family Move for the week: Put the card on the fridge and use it once.”
- Preview next week: “We’ll learn how systems and automation work — how ‘smart’ can come from simple steps.”

## Take-home vocabulary
- Tool
- Job
- Choice
- People-built
- Interface (optional)

## Facilitator notes
- Keep it playful; avoid fear-based examples.
- If a parent starts venting about screen struggles, validate briefly and redirect to: “What job is the tool doing? What boundary would help?”
